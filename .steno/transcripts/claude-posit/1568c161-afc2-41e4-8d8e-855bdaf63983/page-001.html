# Introduction

This workflow implements the DADA2 pipeline for processing 16S rRNA gene amplicon sequencing data. DADA2 infers exact amplicon sequence variants (ASVs) from high-throughput amplicon sequencing data, providing single-nucleotide resolution.

## Pipeline Overview

1. Filter and trim raw sequences
2. Learn error rates
3. Dereplicate sequences
4. Infer sample composition
5. Merge paired-end reads
6. Construct sequence table
7. Remove chimeras
8. Assign taxonomy
9. Export results
